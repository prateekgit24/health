# HOW - Health Over Wealth

HOW is a health and fitness knowledge and planning platform.

It combines:
- verified nutrition and activity facts
- practical routines and goal-based plans
- calculators (BMI, BMR, TDEE, body fat estimations)
- educational guidance for sustainable fitness habits
